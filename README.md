# Battery Health Prediction

## Goal
Predict smartphone battery health using machine learning.

## Dataset
Description of features and targets.

## Models Tested
- Linear Regression
- Ridge Regression
- Decision Tree
- Random Forest
- Gradient Boosting

## Evaluation Metrics
- R²
- RMSE
- MAE

## Experiment Tracking
MLflow was used to track parameters, metrics, and trained models.

## Results
Best model: Gradient Boosting
R²: 0.94
